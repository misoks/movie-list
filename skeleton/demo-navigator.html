Navigator Basics

Each of these cards is a navigator section. By default, each section is in a read-only mode. But by either clicking on it or its corresponding link in the navigation pane to the left, it will transform into an editable mode where any fields with the .editable class it contains will turn into inputs. For example:

If you have more sections than will fit on the screen, the links in the navigation pane will scroll the relevant section to the top of the screen.

Implementation

View the HTML source code for this page to get an idea of how to implement a navigator in Skeleton. The content of the navigation pane is automatically populated based on the content here in the main section. We use the name attribute of the .navigator__section to make the text of the link in the navigation pane, so the header of the actual section can be a little more verbose if you want.
